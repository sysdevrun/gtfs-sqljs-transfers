# gtfs-sqljs-transfers

**[Live demo](https://sysdevrun.github.io/gtfs-sqljs-transfers/)**

A React web app for visualizing transfer connections between Car Jaune bus routes in Réunion Island. Select an arrival route/stop and a departure route/stop, then view a timeline of all connections for a given date with configurable transfer intervals.

Built with [gtfs-sqljs](https://github.com/sysdevrun/gtfs-sqljs) for in-browser GTFS data loading and querying via sql.js (SQLite compiled to WebAssembly).
